# Learn Python by Doing with 100 Projects

This repository contains my work from the Udemy course **Learn Python by Doing with 100 Projects**.

## Goals

- Build practical Python skill through project-based repetition
- Keep all 100 projects organized and searchable
- Practice PyCharm Pro, Git, virtual environments, and clean project layout
- Track lessons learned, bugs fixed, and portfolio candidates
- Upgrade selected projects into portfolio-ready work

## Project Naming Schema

Each project folder should use this format:

```text
s##_p###_short_project_name
```

Examples:

```text
s01_p001_add_date_to_filenames
s01_p002_rename_filenames
s02_p003_reverse_text
s02_p004_sentence_case_converter
```
